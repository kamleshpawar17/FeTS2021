# Federated Tumor Segmentation Challenge 2021 (task-2 2nd place solution)

## Installation
clone the repository and run the setup.sh file
````
sh setup.sh
````

## Data Prepration
```
generate_numpy_data.py
```

## Training
````
cd ./trainingScripts
sh train.sh
````

## Inference
````
inference.py
````
